About


TechNOLEgy is the first Recognized Student Organization at Florida State University focused on making computer programming a social and interdisciplinary experience. We believe every student at Florida State University should learn how to program a computer because it teaches one how to be an innovative and resourceful problem solver.

Computer Programming

We seek to give every student at the Florida State University an opportunity to learn computer programming in a fun, informal, and innovative environment.

Interdisciplinary Collaboration

We hope to foster interdisciplinary discussion and collaboration by motivating all majors to apply for membership, regardless of coding experience.

Entrepreneurial Spirit

We embrace the entrepreneurial spirit at FSU by developing technology from the ground up, from learning how to code to marketing technology.
